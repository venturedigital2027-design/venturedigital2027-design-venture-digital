# Client Delivery Checklist

## 1. Review Intake
- [ ] Identify business type
- [ ] Identify current software
- [ ] Identify biggest bottleneck
- [ ] Identify urgency
- [ ] Identify budget fit

## 2. Define Automation
- [ ] Trigger
- [ ] Inputs
- [ ] Processing steps
- [ ] Outputs
- [ ] Human approval points
- [ ] Failure handling

## 3. Build Manually First
- [ ] Draw workflow
- [ ] Confirm with client
- [ ] Build first version
- [ ] Test with sample data
- [ ] Document handoff

## 4. Deliver
- [ ] Send walkthrough video
- [ ] Send written SOP
- [ ] Confirm client can use it
- [ ] Ask for testimonial
